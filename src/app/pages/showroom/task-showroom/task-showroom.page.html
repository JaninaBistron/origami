Navigationsaufgaben

Bei einer Navigationsaufgabe muss der Spieler zu einem neuen Ort navigieren. Es gibt vier Arten der Navigation:

Navigation zur Zielfahne Platz für ein Bild Der Spieler muss sich mittels Karte zum nächsten Zielpunkt begeben. Der Zielpunkt ist auf der Karte markiert. Navigation mit Richtungspfeil Der Spieler wird von einem Richtungspfeil und mithilfe einer Entfernungsangabe in Metern zum nächsten Zielpunkt geleitet. Die Karte wird ausgeblendet. Navigation mit Textanweisung Der Spieler wird mittels Textanweisung zum nächsten Zielpunkt geleitet. Themenaufgaben

Themenaufgaben sind stets ortsgebunden. Es gibt vier Arten, die hinsichtlich ihres Informationsgehalts nochmal weiter unterschieden werden können:

Selbst-Lokalisation Platz für ein Bild Text Objekt-Lokalisation Platz für ein Bild Text Richtungsbestimmung Platz für ein Bild Text
